Video Deepfake Detection Dataset

deepfake-video-9.6k

Data CardCode (0)Discussion (0)Suggestions (0)

About Dataset

This dataset is a multilingual deepfake video classification dataset, inspired by the structure and methodology of MAVOS-DD. It is designed for binary classification (Real vs Fake) tasks and optimized for ease of use in machine learning pipelines.

Unlike the original dataset—which organizes data by language and generation methods—this version restructures all data into a simplified and unified format, making it highly practical for training deep learning models.

The dataset includes videos from multiple languages such as:

English Hindi German Arabic

All samples are consolidated into a single classification structure:

Dataset/ ├── Real/ └── Fake/

Purpose

This dataset is intended for:

Deepfake detection research Binary video classification (Real vs Fake) Cross-lingual model generalization Training CNN / Transformer / Multimodal models

Inspiration

This dataset takes reference from:

MAVOS-DD

That dataset provides a large-scale multilingual benchmark with:

250+ hours of video Multiple languages Several deepfake generation methods

However, its structure is complex (language → method → files). This Kaggle version simplifies it into a model-friendly format.

Key Transformation:

All real/ folders → merged into Real/ All generated methods (inswapper, memo, roop, etc.) → merged into Fake/ Language distinctions are implicitly preserved in content, not folder hierarchy

Labels

Label Description
Real Authentic, unaltered videos
Fake Deepfake or AI-generated/manipulated videos

Languages Covered

Arabic English Hindi German

(Combined into a unified dataset for generalization)

Features

Each sample corresponds to:

Video file (.mp4) Binary label (Real / Fake)

Inspired metadata fields from MAVOS-DD include:

Language Generation method (collapsed into Fake) Source video
